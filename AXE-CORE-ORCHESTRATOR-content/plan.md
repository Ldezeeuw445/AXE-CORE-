# AXE Command Center — JARVIS-Style AI Operating System

## Overview
Build a futuristic AI Operating System (not a dashboard, not a website) — a JARVIS-style command center for the AXE ecosystem. Central holographic AI core, modular panels, agent system, trading hub, finance hub, MCP center, memory visualization, and more.

## Design Direction
- Iron Man JARVIS + Apple HIG + Linear.app + Tesla UI + Arc Browser
- Color: Matte Black / Deep Charcoal / Dark Navy / Cyan accents
- Style: Glassmorphism, rounded corners, thin borders, subtle shadows, micro-animations
- Centerpiece: Animated holographic sphere/neural core that breathes/pulses/rotates
- Premium, cinematic, minimal, luxury — no cyberpunk, no neon overload

## Skill: vibecoding-webapp-swarm (React + TypeScript + Tailwind + shadcn/ui)

## Stages

### Stage 1 — Design System & Project Setup
- Load vibecoding-webapp-swarm skill

### Stage 2 — Core Layout & Shell
- Top Navigation bar

### Stage 4 — Integration & Polish
